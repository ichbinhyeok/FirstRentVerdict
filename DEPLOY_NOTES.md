# 배포 및 기능 업데이트 내역 (2026-02-04)

## 1. 공유 기능 개선 (Sharing Fix)
- **문제점**: 기존 결과 페이지가 `POST` 요청만 허용하여, 링크를 공유하거나 복사해서 새로 열면 `GET` 요청이 되어 메인 페이지로 리다이렉트되는 현상이 있었습니다.
- **해결**: 결과 페이지를 `GET` 방식(파라미터 기반)으로도 접근할 수 있도록 컨트롤러를 수정했습니다.
- **사용법**: 결과 페이지 하단의 'Copy Link' 버튼을 클릭하면 `.../verdict?cityState=Boston|MA&monthlyRent=2000...` 형태의 링크가 생성되며, 이를 통해 누구나 동일한 계산 결과를 바로 확인할 수 있습니다.

## 2. 404 오류 해결 (Deployment Fix)
- **문제점**: 배포 환경(Docker/Linux)에서 JTE 템플릿 파일이 JAR에 포함되지 않아, `/verdict/boston-ma` 등 서브 페이지 접근 시 404 오류가 발생했습니다.
- **해결**: `build.gradle` 설정을 수정하여 JTE가 생성한 자바 클래스 파일들이 배포용 JAR에 명시적으로 포함되도록 강제했습니다.
- **결과**: 재배포 시 404 오류가 사라질 것입니다.
